## plugin
1. Material Icon Theme
2. Prettier - Code formatter
3. Vue - Official
4. Vue VSCode Snippets
5. ESLint
6. Path Intellisense
7. Auto Import

## VUE CLI 설치
```
npm install -g @vue/cli
```
## 프로젝트 생성
```
vue create vue3-cli
```
## yarn 설치
```
npm install -g yarn
```
## 프로젝트 실행
```
프로젝트 root dir까지 이동 (package.json 있는곳까지)
yarn serve
```
## axios 설치
```
yarn add axios
```
